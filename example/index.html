Google Analytics Widget

The Google Analytics Widget is made to give the user a good feedback when deactivate Google Analytics Example

Download script

add script to page



<script type="text/javascript" src="gaSecraWidget.js"></script>

add CSS class to link


    <!--
    You have the possibility to add a text to give the user help over an "data-text" attribute.
    the class "ga-secra-widget" is needed, because otherwise the widget will fail.
    -->
    <a href="#" data-text="here you can tell why you need Google Analytics"
                class="ga-secra-widget">Disable Google Analytics</a>
